# Queue

La cola (Queue) es una estructura de datos donde los elementos salen en el mismo orden en que entraron.
Mientras una pila usa:

**LIFO**
Último en entrar, primero en salir

Una cola usa:
**FIFO**
Primero en entrar, primero en salir

## Operaciones

- Operación enqueue ¿Qué hace?
- Agrega un elemento al final de la cola.

```dart
queue.add("value");
```

- Operación dequeue ¿Qué hace?
- Elimina el elemento que está al frente.

```dart
queue.removeFirst();
```
