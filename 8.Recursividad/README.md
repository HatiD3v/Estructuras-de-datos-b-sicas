# Recursividad

La recursividad es una tecnica donde una funcion se llama a si misma para resolver un problema y es fundamental para entender:

- Arboles
- Grafos
- Busquedas Avanzadas
- Algoritmos complejos

## Idea central

Imaginemos que tenemos una tarea muy grande.
En lugar de resolverla de una sola vez.

1. Se divide en versiones mas pequenas
2. Se resuelve esa version mas pequena
3. Repetir hasta llegar a un caso muy simple.

Eso es recursividad.

### Componentes obligatorios

Toda funcion recursiva tiene:

1. Caso base: condicion que detiene la recursion
2. Llamada recursiva: La funcion se llama a si misma
